# 🍳 RecEngine | Smart Recipe Finder

RecEngine is a modern, responsive web application designed to help users discover recipes based on the ingredients they already have. With a database of **100+ recipes**, it simplifies meal planning through an intuitive "Pill-based" filtering system.

🚀 **[Live Demo](https://mertkx.github.io/RecEngine/)**

---

## ✨ Key Features

- **Smart Filtering:** Select multiple ingredients to see which recipes you can cook right now.
- **100 Recipes Database:** A wide variety of Turkish and International cuisines.
- **Difficulty Visualization:** Color-coded cards (Green: Easy, Yellow: Medium, Red: Hard).
- **Interactive Modal:** View detailed cooking instructions and ingredients in a focused popup.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.
- **Fast Search:** Real-time search bar for finding specific recipes or ingredients.

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3 (Custom Variables, Flexbox, Grid)
- **Scripting:** Vanilla JavaScript (ES6+)
- **Data:** JSON-based local API
- **Deployment:** GitHub Pages
